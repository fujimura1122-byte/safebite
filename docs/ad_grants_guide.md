# Google for Nonprofits (Ad Grants) 申請ロードマップ
## SafeBite — 毎月1万ドルの無料検索広告枠を獲得する手順書

> **概要**: Google Ad Grants は、認定NPO団体に対して Google 検索広告枠を毎月 **最大1万USD（約150万円相当）** 無料で提供するプログラムです。SafeBiteが「闇バイト」「やみバイト」などのキーワードで検索広告を出稿できれば、最も危険な状態のユーザー（今まさに闇バイトを探している人）に直接リーチできます。

---

## ロードマップ全体像

```
Phase 1: 法人格の取得（2〜3ヶ月）
    ↓
Phase 2: TechSoup Japan へのNPO登録（1〜2週間）
    ↓
Phase 3: Google for Nonprofits への申請（審査1〜2週間）
    ↓
Phase 4: Google Ad Grants の有効化（即日〜3営業日）
    ↓
Phase 5: SafeBite サイトの審査通過要件を整備（並行作業）
    ↓
Phase 6: 広告キャンペーンの設定・運用開始
```

## Phase 1: 法人格の取得

### なぜ法人格が必要か

Google for Nonprofits の審査において、**個人ではなく法人（団体）** であることが必須条件です。

### 推奨: 一般社団法人の設立

NPO法人（特定非営利活動法人）は設立に10名以上の社員と2〜3ヶ月の行政審査が必要ですが、**一般社団法人**は2名から設立でき、登記だけで完了します（設立費用: 約6万円）。

| 比較項目 | NPO法人 | 一般社団法人（推奨） |
|---|---|---|
| 設立人数 | 10名以上 | **2名以上** |
| 設立期間 | 2〜3ヶ月 | **1〜2週間** |
| 費用 | 約1万円 | **約6万円** |
| Google認定 | ◎ | ◎ |

### 手順

1. **定款の作成**: 目的に「インターネット上の危険情報から若者を守る啓発活動」を明記
2. **公証役場で定款認証**: 不要（一般社団法人は認証不要）
3. **法務局で設立登記申請**: 登録免許税6万円
4. **法人設立完了**: 登記簿謄本を取得

> **費用を抑えたい場合**: 設立登記の司法書士費用（3〜5万円）の節約のため、「freee会社設立」などのサービスを活用できます。

## Phase 2: TechSoup Japan へのNPO登録

### TechSoupとは

Google、Microsoft、Adobeなどのテクノロジー企業が、NPO団体を認定するために委託した第三者機関です。TechSoupの認定を受けることで、複数のIT企業の非営利支援プログラムへの申請資格が得られます。

### 登録手順

1. **TechSoup Japan にアクセス**  
   https://www.techsoupjapan.org/

2. **アカウント作成**  
   「新規登録」から団体情報を入力

3. **提出書類**（一般社団法人の場合）  
   - 登記事項証明書（発行3ヶ月以内）
   - 定款のコピー
   - 代表者の身分証明書

4. **審査期間**: 1〜2週間

5. **認定通知を受信**: TechSoupから認定メールが届きます

## Phase 3: Google for Nonprofits への申請

### 申請手順

1. **Google for Nonprofits にアクセス**  
   https://www.google.com/nonprofits/

2. **「今すぐ申請」をクリック**

3. **必要情報を入力**  
   - TechSoup のバリデーショントークン（TechSoup認定後に発行）
   - 団体のGoogleアカウント（Gmail、Google Workspaceどちらでも可）
   - ウェブサイトURL: `https://safebite-zeta.vercel.app`

4. **Google の審査**: 1〜2週間

5. **承認通知を受信**

### 重要: 審査落ちする団体の条件（SafeBiteが避けるべきこと）

Google は以下の団体を審査対象外としています：

- 政府機関・政府系団体
- 病院・医療機関
- **営利目的の活動** ← アフィリエイト収入があっても、「主目的が公益である」ことを明確にすれば問題なし
- 個人や家族を主な受益者とする団体

## Phase 4: Google Ad Grants の有効化

Google for Nonprofits 承認後、ダッシュボードから **「Google 広告」** の欄で Ad Grants を有効化します。

- Google 広告アカウントが自動作成されます
- 月1万USDの無料クレジットが付与されます
- **CPC（クリック単価）上限: 2USD** という制約あり（後述）

## Phase 5: SafeBiteサイトの審査通過要件の整備（今すぐ着手）

**Ad Grants 審査・維持のためにサイトに必要な要件**をまとめます。現状のSafeBiteに不足している箇所と、必要な追加対応を示します。

---

### 必須要件 1: プライバシーポリシーページの設置

**現状**: なし → **要追加**

Ad Grants の審査において、独立したプライバシーポリシーページは必須です。

**追加するページ**: `/privacy`

**記載必須事項**:
```
・収集する情報の種類（SafeBiteは入力テキストを保存しないことを明記）
・Googleアナリティクスによるアクセス解析の実施
・Cookieの使用
・第三者への情報提供なし
・お問い合わせ先
・運営団体名・所在地
```

---

### 必須要件 2: ミッション・活動実績ページの設置

**現状**: `AboutSection`（トップページ内の1セクション）→ **独立ページに格上げ要**

Ad Grants の審査官は「この団体が本当に非営利活動をしているか」を確認します。独立した `/about` ページに以下を記載してください。

**記載内容**:
```
・団体の設立目的・ミッション
・活動実績（AI危険度チェック累計〇〇件、IHC通報支援〇〇件など）
・代表者プロフィール（任意だが信頼性が上がる）
・運営団体の法人情報
・報道実績・外部リンク（あれば）
・お問い合わせ方法
```

---

### 必須要件 3: 問い合わせページの設置

**現状**: なし → **要追加**

**追加するページ**: `/contact`

メールアドレスまたはフォームを設置してください。  
（Google フォームへの外部リンクでも審査通過例あり）

---

### 必須要件 4: サイト品質基準の確認

| チェック項目 | 現状 | 対応 |
|---|---|---|
| HTTPS | ✅ Vercelで対応済み | 対応不要 |
| モバイル対応 | ✅ Tailwindで対応済み | 対応不要 |
| ページ読み込み速度 | ✅ Next.js静的生成 | 対応不要 |
| サイト独自ドメイン | ⚠️ vercel.app サブドメイン | **カスタムドメイン推奨** |
| プライバシーポリシー | ❌ なし | **要追加** |
| 問い合わせページ | ❌ なし | **要追加** |
| ミッションページ | ⚠️ トップページ内のみ | **独立ページ化推奨** |

> **独自ドメインについて**: `safebite.jp` などの取得を推奨します（年間約1,200円）。Ad Grants の審査上の必須要件ではありませんが、広告の品質スコア向上・信頼性向上・SEOに有効です。

## Phase 6: Ad Grants 広告キャンペーンの設定

### ターゲットキーワード例（闇バイト関連）

| キーワード | 月間検索数（推定） | 意図 |
|---|---|---|
| 闇バイト 見分け方 | 高 | 情報収集層 |
| 闇バイト 断り方 | 中 | 危機意識あり層 |
| 闇バイト 逃げたい | 中 | **最優先** - 今すぐ助けが必要 |
| 闇バイト 通報 方法 | 中 | 行動意欲あり |
| UD 意味 バイト | 低〜中 | 隠語を調べている層 |
| 受け子 バイト | 低〜中 | 隠語調査層 |
| 即金 バイト 危険 | 高 | 注意喚起層 |
| 日払い バイト 怪しい | 高 | 疑念を持った層 |

### Ad Grants の制約と対策

| 制約 | 内容 | 対策 |
|---|---|---|
| CPC上限 | 1クリック最大2USD | キーワードを具体的にしてQuality Scoreを上げる |
| CTR維持 | 月間CTR 5%以上必要 | 広告文を定期的に改善する |
| サイト品質 | 低品質サイトは停止 | 各ページにオリジナルコンテンツを充実させる |
| 単一キーワード | 1語キーワード不可 | 「闇バイト 見分け方」など2語以上を使う |
